# SPA Example

`index.html` is a no-build browser example.

It imports `@dawdle.space/bookshelf` from a CDN, calls the hosted bookshelf API, and renders the returned data client-side. Use this when you want a simple page that stays up to date without adding server-side code.

If you do not need continuous updates, use the static export or CLI instead. That gives you a lighter site because the final page only contains CSS and generated HTML.
